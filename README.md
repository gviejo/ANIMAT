SOFTWARE VERSIONS
-------------------------------

 * boost (minimum version 1.46)			
 * boost-thread
 * boost-program_options
 * boost-system
 * boost-filesystem
 * SDL (minimum version 1.2)
 * GSL (minimum version 1.15)
 * cmake (minimum version 2.8.7)

	
INSTALLATION 
-------------------------------

	git clone https://github.com/gviejo/ANIMAT.git
	cd Projet*
	cmake .
	make
